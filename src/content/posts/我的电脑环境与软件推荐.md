或许你刚刚结束高中生涯，得到了人生中第一台电脑，或者你已经是计算机专业的大学生，来寻求共鸣，虽然我是计算机专业的学生，但本篇博客并不局限于计算机专业，尽管多数软件是面向计算机专业的学生，但我的计算机使用理念是面向所有人的。

# 配置选择与系统

电脑是一台工具，选择电脑配置时应根据自己的需求来选择。网络上有很多电脑配置推荐，兼听则明，博采众长，选择适合自己的配置。这里我不会给出具体推荐，你应该学会查阅资料，搜索教程，这也是接触电脑的第一步。

系统的选择，你或许听说过 Windows、macOS、Linux 甚至 BSD 等系统，根据你购买的电脑类型，它们一般已经预装了系统。一般来说，我并不推荐你去更换系统，除非你对操作系统有一定的了解，或者你有特殊需求。不要因为在网上看到 Linux 美化的桌面截图就想去安装 Linux，或是听到有人对 Windows 的吐槽就想去安装黑苹果。

如果你是第一次接触电脑，那么你最重要的任务是熟悉电脑的使用，了解你会用电脑来做些什么。在这个过程中，你会走很多弯路，遇到很多问题，我的系统环境也不是一开始就配置好的，也是在使用过程中不断调整的。你可以参考我的电脑环境，但不要照搬，适合自己的才是最好的。

我选择了 Windows 11 作为我的主要操作系统，稳定、兼容性好、软件丰富，适合大多数人使用。作为计算机学生，我也安装了 WSL 作为编程与学习 Linux 的环境，这会在后面进行详细介绍。

# 软件安装

如果你只用过手机，希望电脑上的微软商店能够满足你的需求，那么，在大多数情况下，这确实是一个不错的选择，然而，微软商店的软件数量有限，如果你发现你需要的软件不在商店中，那么你就需要去官网下载，或者使用其他软件管理工具来安装软件。

对于计算机学生，包管理器是非常实用的工具，它可以帮助你快速安装、更新和卸载软件。Windows 上主要有Chocolatey、Scoop 和 winget 三种包管理器，我个人使用的是 winget和 Scoop，winget 是微软官方的包管理器，本质上就是在下载安装包并执行安装命令，而 Scoop 则可以进行绿色安装，极大程度的减少了对系统的污染。当然，由于这里不是 linux 系统，不能奢望所有软件都能通过包管理器安装，部分软件仍然需要去官网下载并安装。

安装软件时，注意安装选项，不要因为少取消了一个勾选框而安装了不需要的软件，或是缺失了必要的组件。~~我知道你懒得看~~

# 系统配置

系统优化是一个很大的话题，网上有很多教程，但你大概率不会一开始就去优化系统，只要不是有洁癖或是希望电脑总是保持最高的效能，大概会等到你使用电脑一段时间后，遇到卡顿或是其他问题时，才会去优化系统。总的来说，系统优化有以下几个方面：

1. 减少开机启动项，关闭不必要的后台程序。
2. 卸载自带的无用软件
3. 关闭不必要的服务
4. 禁止系统自动更新~~微软罪大恶极~~
5. 磁盘清理与碎片整理

关于自动更新，微软的自动更新机制确实让很多人头疼，并且大部分人确实不需要那些新功能，然而，关闭自动更新也有风险，尤其是安全更新，你也不想因为关闭了自动更新而导致电脑中毒吧。我的建议是，如果要关闭自动更新，最好关注一下Windows爆出的严重漏洞。

如果你使用的是笔记本电脑，电源管理也是一个重要的方面，尤其是对于游戏本来说，要是你用节能模式玩游戏，或是用狂暴模式办公，或多或少都会影响电脑的性能和续航，检查一下电源管理设置或自带的管理软件，选择适合的模式。

# 系统美化

我相信很多人看到网上的 Windows 美化教程后，都会想去尝试一下，然而，我并不推荐一上手就去美化系统，尤其是对于新手来说，跟随美化教程，你大概率会下载一堆你并不了解的软件，不清楚它们的作用乃至对系统的影响，最终真正美化后，你可能会发现很多美化事实上牺牲了易用性，甚至会导致系统性能下降或不稳定。不是总有人说，“美化的尽头是默认”吗？我认为这句话是有道理的，不过，尝试折腾系统也是一种乐趣，但如果你并不享受这种乐趣，那最好还是秉持“能跑就行”的原则。

# 软件推荐

这里主要推荐一些我认为比较实用的软件，至于一些不得不用的软件比如微信、office等，我就不再赘述了。

## 1. 视频类

视频分为网页视频和本地视频两类，在电脑上，浏览器观看网页视频是最常见的方式，至于网站对应的客户端软件，我并不看好，一方面，客户端软件可能就是一个网页浏览器的壳子，另一方面，客户端难以自定义，且可能会占用更多的系统资源。当然，两种方式并没有绝对的优劣，方便且易上手的客户端软件也有其存在的价值。

本地视频播放软件就百家争鸣了，VLC、PotPlayer、MPV和自带的电影与电视都是不错的选择，这里推荐以下：

1. **VLC**：开箱即用，支持几乎所有视频格式，功能强大，适合新手使用。界面简洁（虽然不够美观~~但是为什么手机端界面那么现代~~）但可以加载自定义皮肤，进阶用户可以通过安装插件来扩展功能。
2. **PotPlayer**：需要一定的配置，名声在外（几乎所有人都在推荐），最最重要的是修改渲染器为 Direct3D 11和关闭声音规格化，同样支持几乎所有视频格式，功能强大，界面复古但同样可以加载自定义皮肤，进阶用户可以安装额外的渲染器和解码器来增强体验（比如 madVR、LAV Filters 等），但体感上区别不大，除非你是视频发烧友。
3. **MPV**：没有可视化配置界面，配置文件是纯文本的，适合喜欢折腾的用户，功能强大，学习成本较高，但拓展性极强，支持几乎所有视频格式，界面简洁（甚至没有界面），可以通过配置文件和脚本实现各种功能，适合喜欢自定义的用户。

我使用的是 MPV，通过使用网络上以及配置好的配置文件，上手难度可以接受，用来看动漫时还能用anime4k提高画质，体验极佳。不过，折腾这家伙还是花了我几天时间，如果你是新手，建议先使用 VLC 或 PotPlayer。

## 2. 密码管理器

为了保护个人隐私和信息安全，密码管理器是非常重要的工具。它解决了安全的密码记不住和记住的密码不安全的问题。可以选择 KeePassXC、Bitwarden、1Password 等密码管理器，我使用的是 Bitwarden，免费功能已经足够使用，支持多平台同步，界面简洁，功能强大。

借此机会，我想强调一下密码安全的重要性。例如，不少人的微软账号被盗，购买的软件和游戏也被盗，虽然也有微软在账号安全方面的责任，但为了保护自己的账号安全，开启2FA（双因素认证）或是passkey 是非常有必要的。同时，使用密码管理器可以帮助你生成强密码，并安全地存储和管理这些密码，避免使用简单或重复的密码，从而提高你的网络安全性。

不过，密码管理器本身的密码也是一个问题，要是密码管理器的主密码被破解了，那就等于所有密码都被泄露了，在设置复杂密码的同时，开启2FA，打印恢复码并妥善保存，都是非常重要的措施。

## 3. 快捷地打开软件或文件

PowerToys 是微软官方的一个工具集，其中的命令面板可以通过搜索快速打开软件或网站与其他操作，再搭配上 Everything（搜索文件）集成到命令面板中，可以大幅提高操作效率。

## 4. 解压和下载器

我使用 nanaZip 作为解压软件，Matrix Next 作为下载器，nanaZip 是一个以 7-Zip 为基础的开源解压软件，界面符合 Windows 11 的设计风格，支持多种压缩格式，功能强大且轻量，智能提取压缩包功能尤为好用。Motrix Next 是一个开源的下载器，协议支持全面，http, bt, ed2k, magnet 等，支持多线程下载，界面简洁，功能强大。

## 5. 浏览器插件

浏览器插件可以扩展浏览器的功能，提高浏览体验。以下是一些我认为比较实用的插件推荐：

1. **uBlock Light**：一款高效的广告拦截插件，支持多种浏览器，能够有效屏蔽广告和恶意网站，提高浏览速度和安全性。
2. **Tampermonkey**：一款用户脚本管理器，支持多种浏览器，可以通过安装用户脚本来扩展浏览器功能，实现个性化的浏览体验。
   - 例如美化网页、自动填写表单、屏蔽弹窗广告、自带翻页等功能。用户可以在 Greasy Fork 等网站上找到各种实用的脚本，满足不同需求。
3. **Bitwarden**：Bitwarden 对应的密码管理器插件，方便在浏览器中自动填写密码，提高安全性和便利性。
4. **Dark Reader**：一款夜间模式插件，支持多种浏览器，可以将网页转换为暗色主题，保护眼睛，减少蓝光伤害。
5. **Motrix Next**：Motrix Next 对应的下载器插件，支持多种浏览器，拦截下载链接，方便快速下载文件，提高下载效率。
6. **BewlyCat**：bilibili 网页美化和功能增强插件，继承BewlyBewly，不过卡顿明显
7. **沉浸式翻译**：一款翻译插件，支持多种浏览器，可以在网页上直接翻译文本，双语对照，方便学习和阅读外文资料。

## 计算机专业软件

计算机专业的软件种类繁多，涉及到编程、开发、设计等多个方面。以下是一些我认为比较实用的软件推荐：

1. **Visual Studio Code**：大名鼎鼎的VS Code，轻量级的代码编辑器，支持多种编程语言和扩展插件，适合初学者和专业开发者使用。
2. **CLion**：JetBrains 出品的 C/C++ IDE，大部分计算机专业学生应该都要学C语言，CLion 做到开箱即用，避免新手往往在配置环境上浪费大量时间
3. **NotePad4**：一款轻量高效的文本编辑器，偶尔修改配置文件或是查看日志时非常方便，支持多种编码格式和语法高亮。不支持插件和标签页，功能简单易用。
4. **Git**：必须掌握的版本控制工具，适用于团队协作和个人项目管理。
5. 命令行补全工具：比如 PSCompletion, zsh + oh-my-zsh 等，可以提高命令行操作效率，让你在使用过程中掌握更多的命令行技巧。
6. **WSL（Windows Subsystem for Linux）**：对于计算机专业学生来说，学习 Linux 是非常重要的，WSL 提供了一个在 Windows 上运行 Linux 的环境，方便进行开发和学习 ~~最好的 Linux 发行版~~ 。我在wsl中安装了 Fedora44，拥有较新的软件包和良好的社区支持，适合学习和开发。（I don't use Arch, btw）
7. **PixPin**：截图工具，可以将截图钉在屏幕上，方便查看和对比，还能识别文字，简单标注~~为什么Latex公式识别要收费呜呜~~
